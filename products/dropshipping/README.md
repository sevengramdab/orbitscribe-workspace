# Dropshipping Product Catalog

This directory holds the live product catalog for the OrbitScribe Dropshipping vertical.

## Files

- `DS-*.json` — Individual product records generated by the DropshippingAgent.
- `catalog.json` — Auto-generated index of all active products (updated by the agent).

## Product Schema

Each product JSON contains:

| Field | Description |
|-------|-------------|
| `sku` | Internal stock-keeping unit (auto-generated) |
| `title` | Product title |
| `category` / `niche` | Product niche/category |
| `description` | Short product description |
| `base_cost` | Supplier unit cost (USD) |
| `shipping_cost` | Estimated shipping cost (USD) |
| `sale_price` | Target selling price (USD) |
| `created_at` | ISO timestamp of creation |
| `agent_id` | ID of the agent that sourced the product |
| `last_price_update` | ISO timestamp of last price change |
| `supplier` | Supplier name (`spocket`, `aliexpress`, `oberlo`, or `simulated`) |
| `mode` | `SIMULATION` or `LIVE` |

## Live Readiness

To enable real supplier API lookups in LIVE mode, set at least one of these in `swarm-backend/.env`:

```bash
SPOCKET_API_KEY=...
ALIEXPRESS_API_KEY=...
OBERLO_API_KEY=...
```

When in `SIMULATION` mode (default), the agent uses realistic synthetic costs and shipping estimates based on niche.
